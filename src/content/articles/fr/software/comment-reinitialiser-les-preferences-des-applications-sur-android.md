Le nom est simplement trompeur. Cette opération ne réinitialise pas les applications elles-mêmes. Elle remet à zéro les choix du système qui concernent les autorisations, les applications par défaut, les notifications, l'activité en arrière-plan et les applications désactivées. Vos photos, messages, fichiers, comptes enregistrés et données d'application restent en place.

## Ce qui est réinitialisé

Depuis plusieurs années, l'écran de confirmation d'Android mentionne les mêmes éléments principaux : les applications désactivées, les notifications d'applications désactivées, les applications choisies par défaut pour certaines actions, les restrictions de données en arrière-plan et les restrictions liées aux autorisations. Sur certaines versions récentes d'Android et certaines interfaces de fabricants, la boîte de dialogue mentionne aussi les réglages d'utilisation de la batterie.

En pratique, voici ce qui peut changer :

- Les applications que vous avez désactivées peuvent être réactivées. Cela concerne notamment des applications préinstallées ou des composants système que vous aviez coupés manuellement.
- Les choix d'applications par défaut sont effacés. Android peut vous demander de nouveau quelle application utiliser pour ouvrir des liens, des PDF, passer des appels, envoyer des SMS ou afficher l'écran d'accueil.
- Les blocages de notifications peuvent être annulés. Des applications que vous aviez rendues silencieuses peuvent recommencer à envoyer des notifications jusqu'à ce que vous les désactiviez de nouveau.
- Les restrictions de données en arrière-plan sont supprimées. Une application à laquelle vous aviez interdit d'utiliser les données mobiles en arrière-plan peut retrouver cet accès.
- Les restrictions liées aux autorisations sont réinitialisées. Les applications peuvent redemander l'accès à l'appareil photo, au micro, à la position, aux photos, aux notifications ou aux appareils à proximité lorsqu'elles en ont besoin.
- Les limites d'utilisation de la batterie peuvent aussi revenir à leur valeur par défaut sur les versions d'Android et les interfaces qui les incluent dans cette opération.

Ce qui n'est pas effacé compte tout autant. L'opération ne désinstalle pas les applications, ne supprime pas leurs données, n'efface pas vos fichiers, vos photos, vos messages ou vos téléchargements, et ne vous déconnecte pas de tous vos services. Si vous étiez connecté à WhatsApp, Instagram, une application bancaire ou Gmail avant la réinitialisation, cette action n'est pas censée vous déconnecter.

C'est précisément son intérêt. Elle remet à zéro des réglages généraux, pas les données.

## Quand cela vaut la peine d'essayer

Utilisez cette réinitialisation lorsque le problème semble toucher plusieurs applications, sans être assez grave pour justifier un rétablissement de la configuration d'usine.

L'ouverture des liens est un bon exemple. Vous appuyez sur un lien et il s'ouvre dans le mauvais navigateur, dans une application que vous n'utilisez pas ou pas du tout. Effacer le choix par défaut d'une seule application peut suffire, mais réinitialiser toutes les préférences est plus rapide lorsque plusieurs associations se sont embrouillées.

Cette option est aussi utile si vous avez désactivé plusieurs applications préinstallées avant de voir apparaître des dysfonctionnements. Les applications Android dépendent parfois de composants discrets dont le nom ne dit rien. Vous en désactivez un, une autre application se met à planter, et le lien entre les deux n'est pas évident. La réinitialisation réactive les applications désactivées, ce qui permet de vérifier si l'une d'elles était nécessaire.

Les notifications constituent un autre cas fréquent. Android a gagné de nombreux réglages par application, dont une autorisation d'envoyer des notifications sur les versions récentes. Si une seule application ne vous avertit plus, corrigez directement ses paramètres. Si plusieurs applications se taisent après une mise à jour, une réinitialisation globale est une étape intermédiaire raisonnable.

Je ne ferais pas cela comme entretien régulier. Réinitialiser les préférences tous les mois n'apporte rien. Utilisez cette option lorsqu'un problème existe réellement.

## Comment réinitialiser les préférences des applications

Le chemin varie entre les Pixel, Samsung, Xiaomi, Motorola, OnePlus et les autres interfaces Android. Le plus sûr est d'utiliser la recherche intégrée aux Paramètres et de saisir « préférences des applications » ou « réinitialiser les applications ».

Sur les Pixel récents, l'option se trouve généralement dans Paramètres > Système > Options de réinitialisation. Ouvrez ensuite l'option consacrée aux préférences des applications et confirmez la réinitialisation.

Sur de nombreux téléphones Samsung Galaxy, commencez par Paramètres > Applications, puis ouvrez le menu à trois points. L'option de réinitialisation des préférences se trouve normalement dans ce menu, mais son intitulé et sa position peuvent changer selon la version de One UI.

Sur les téléphones Xiaomi, Redmi et POCO, commencez par Paramètres > Applications > Gérer les applications, puis vérifiez le menu à trois points. Là encore, le chemin exact dépend de la version du logiciel.

Sur d'autres appareils, l'option peut se trouver dans la page Applications, derrière un menu à trois points, ou dans Système > Options de réinitialisation. Utilisez la recherche des Paramètres si vous ne la voyez pas.

La réinitialisation prend effet immédiatement. Il n'est pas nécessaire de redémarrer, même si un redémarrage après l'opération ne pose aucun problème lorsque vous tentez de résoudre un dysfonctionnement tenace.

## Ce qui se passe ensuite

Pendant un jour ou deux, votre téléphone peut vous poser davantage de questions. Appuyez sur un lien, et Android peut vous demander quel navigateur utiliser. Ouvrez un PDF, et il peut vous proposer Drive, Acrobat, votre navigateur ou une autre application. Lancez une application qui utilise l'appareil photo ou la position, et une demande d'autorisation peut réapparaître.

C'est normal.

Le côté agaçant consiste à refaire les choix que vous aviez faits volontairement. Si vous aviez désactivé des applications de l'opérateur, des galeries en double, des navigateurs supplémentaires ou d'autres logiciels préinstallés, retournez dans Paramètres > Applications pour les désactiver de nouveau. Si vous aviez réglé finement les notifications, vérifiez les applications importantes. Si vous limitiez les données mobiles des réseaux sociaux ou des applications vidéo, contrôlez aussi ces réglages.

Voilà pourquoi cette réinitialisation est sûre, mais pas invisible. Elle ne détruit pas vos données, mais elle annule certains ajustements.

## Différences avec les autres types de réinitialisation

Vider le cache d'une application supprime ses fichiers temporaires. Essayez cette solution lorsqu'une seule application est lente, instable ou affiche des données anciennes.

Effacer le stockage d'une application va plus loin. Cette action remet à zéro les données internes de l'application, peut vous déconnecter et supprimer ses réglages locaux. Utilisez-la avec prudence.

La réinitialisation des préférences se situe entre les deux. Elle touche toutes les applications, mais uniquement leurs réglages gérés par le système. C'est ce qui la rend utile lorsque plusieurs petits problèmes apparaissent et qu'aucune cause unique ne se dégage.

Le rétablissement de la configuration d'usine est tout autre chose. Il efface le téléphone et repart de la version d'Android actuellement installée. N'en arrivez pas là simplement parce que les liens s'ouvrent dans la mauvaise application.

## Questions fréquentes

Cette opération va-t-elle me déconnecter des applications ? Non, pas à elle seule. Elle n'efface ni le stockage des applications ni les données des comptes.

Peut-elle corriger une consommation excessive de batterie ? Parfois, indirectement. Si un composant système désactivé ou une autorisation trop limitée obligeait des applications à répéter des tâches en arrière-plan, la réinitialisation peut aider. Si la batterie est usée ou qu'une application consomme réellement beaucoup d'énergie, elle ne fera pas de miracle.

Devrai-je accorder de nouveau des autorisations ? Oui, pour certaines applications. Elles pourront les redemander lorsque l'accès sera nécessaire.

Faut-il essayer avant un rétablissement de la configuration d'usine ? Oui, si le problème concerne le comportement des applications et non un téléphone verrouillé, une corruption grave ou une panne matérielle. L'opération prend environ une minute et n'efface pas votre vie numérique.
