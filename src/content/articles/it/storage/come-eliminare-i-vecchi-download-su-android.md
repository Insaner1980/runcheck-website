Apri un PDF una sola volta, scarichi un APK per fare una prova, salvi un file ZIP dal browser e poi dimentichi tutto. Sei mesi dopo, la cartella Download è piena di file che non riconosci.

È normale. Android mette lì molti file usati una volta sola e quasi nessuno tratta quello spazio come qualcosa da mantenere in ordine. Invece andrebbe fatto.

## Dove finiscono di solito i download

Sulla maggior parte degli smartphone Android, il file manager mostra la cartella pubblica con il nome **Download** o **Downloads**. Sotto la superficie corrisponde spesso a un percorso come `/storage/emulated/0/Download`, ma non devi ricordarlo a meno che tu non usi ADB, un file manager da computer o un'app più avanzata.

Browser come Chrome e Firefox salvano lì i file per impostazione predefinita. Anche gli allegati delle email possono finirci quando scegli di salvarli invece di visualizzarli soltanto. Alcune app creano sottocartelle proprie dentro Download. Altre depositano i file direttamente lì, spesso con nomi per niente chiari.

Non tutti i contenuti scaricati si trovano in quella cartella. App di messaggistica, podcast, mappe e servizi cloud spesso usano uno spazio specifico dell'app o cartelle multimediali proprie. Documenti di WhatsApp, file di Telegram ed episodi di podcast offline possono occupare molta memoria anche quando Download sembra vuota.

## Inizia dal file manager

Apri l'app per i file fornita con il telefono. Sui Pixel e su molti dispositivi con Android vicino alla versione standard è Files by Google. Sui Samsung di solito è Archivio.

Vai alla categoria o alla cartella Download. Ordina prima per data. I file vecchi sono le occasioni più semplici, perché è raro che il telefono abbia bisogno di qualcosa che hai scaricato otto mesi fa e non hai più aperto.

Poi ordina per dimensione. È lì che si recupera davvero spazio. Una presentazione può pesare 80 MB. Un video scaricato può arrivare a 600 MB. I vecchi file APK di giochi grandi possono essere molto più pesanti del previsto.

Non eliminare tutto senza guardare. Ma sii realistico: la maggior parte delle cartelle Download non è un archivio. È uno spazio temporaneo che nessuno ha mai ripulito.

## Cosa puoi eliminare senza problemi

Di solito puoi rimuovere i file APK dopo aver installato l'app. L'APK è il pacchetto di installazione, non l'app già installata. Se proviene da un sito web, conservalo soltanto se ti serve di nuovo proprio quella versione.

PDF, immagini, documenti e fogli di calcolo si possono eliminare quando l'originale è ancora facile da recuperare. Gli allegati delle email in genere restano nel messaggio. I file scaricati da un sito spesso possono essere scaricati di nuovo.

Anche i file ZIP e gli altri archivi sono buoni candidati, soprattutto se li hai già estratti. Quando il contenuto si trova in un'altra cartella, l'archivio originale spesso è soltanto peso morto.

Fai più attenzione a ciò che sembra un backup, un'esportazione, un documento fiscale, un documento di viaggio o un file di recupero. Sono proprio questi i file che rendono costosa una pulizia distratta.

## Usa i suggerimenti di pulizia

Files by Google ha una sezione Pulisci che può proporre i file scaricati da esaminare. Apri l'app, vai su **Pulisci** e cerca la scheda **Elimina i file scaricati**. Google non la chiama necessariamente "vecchi download".

Controlla l'elenco prima di spostare i file nel Cestino. L'app invia prima gli elementi selezionati al Cestino, quindi di solito hai un periodo in cui puoi recuperarli. Non considerare però il Cestino un sistema di backup. Se un file conta, spostalo su Drive, Foto o in un'altra posizione di backup prima di rimuoverlo dal telefono.

Su Samsung puoi fare una pulizia simile da Archivio. Apri **Archivio > Categorie > Download**, seleziona i file ed eliminali. Sui Galaxy più recenti Samsung offre anche strumenti di pulizia da **Impostazioni > Assistenza dispositivo > Spazio di archiviazione** o **Impostazioni > Assistenza dispositivo e batteria > Memoria archiviazione**, a seconda del modello e della versione di One UI.

## Controlla questi elementi prima di eliminare in massa

I nomi casuali non indicano sempre spazzatura. Alcune app salvano file con hash o lunghe sequenze di numeri. Se l'estensione è chiara, per esempio `.jpg`, `.pdf`, `.mp4` o `.apk`, puoi farti un'idea. Se il file non ha estensione o si trova in una cartella con il nome di un'app, rallenta.

I file ancora aperti possono rifiutarsi di essere eliminati. Chiudi l'app che li sta usando e riprova.

Controlla anche la sincronizzazione cloud. Eliminare un file locale da una normale cartella Download di solito non cancella la copia originale nel cloud, ma rimuovere un elemento dentro una cartella sincronizzata può interessare anche la versione online. Google Drive, OneDrive, Dropbox e servizi simili non si comportano tutti allo stesso modo.

## Evita che la cartella si riempia di nuovo

Chrome per Android può chiedere dove salvare un file prima di scaricarlo. Apri **Chrome > Impostazioni > Download** e attiva **l'opzione che chiede dove salvare i file**, se è disponibile nella tua versione. Quella piccola pausa è utile. Ti costringe a decidere se il file deve finire in Download, su Drive o da nessuna parte.

Vale la pena controllare anche le app di messaggistica. In WhatsApp apri **Impostazioni > Spazio e dati > Download automatico media** e disattiva i tipi di file che non vuoi salvare automaticamente. Telegram e Signal offrono controlli simili, anche se i nomi esatti cambiano.

Un'abitudine semplice funziona meglio di un sistema complicato: ogni pochi mesi ordina Download per data, elimina ciò che è chiaramente inutile, poi ordina per dimensione e rimuovi i file grandi che non ti servono. Per la maggior parte delle persone basta questo.
